## 📱 **Escenarios de Demo**

### **Escenario 1: Problema Urgente de Cliente**

1. Cliente reporta falla en equipo CNC
2. Crea solicitud urgente con fotos
3. Sistema asigna agente automáticamente
4. Chat en tiempo real para resolución
5. Usa calculadora para verificar parámetros
6. Agente resuelve y cliente califica

### **Escenario 2: Consulta de Ventas**

1. Cliente busca información de productos
2. Contacta agente de ventas via directorio
3. Chat de ventas con cotización
4. Usa calculadora para evaluar opciones
5. Crea solicitud formal de compra

### **Escenario 3: Supervisión Administrativa**

1. Admin revisa métricas diarias
2. Ve solicitudes pendientes
3. Asigna agentes según carga de trabajo
4. Monitorea satisfacción del cliente
5. Genera reportes ejecutivos
